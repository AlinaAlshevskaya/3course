MathCAD PDF Creation Release Notes

 

Welcome to the release notes which contain important notes, system requirements, and information on issues you may encounter when using MathCAD PDF Creation for Windows.

 

Critically important

 

To install, the user must be logged in as an Admin and the silent installation method must be used.  Below is the command line to use from a calling program or script.

 

msiexec /i "<path to Distillr.msi> ISX_SERIALNUMBER="1071-1006-8094-6401-2690-6767" TRANSFORMS="<path to the MathCad.mst>” /qn

Windows System Requirements

 

MathCAD PDistiller 9 for Windows® requires the following:

 

·         1.3 gigahertz processor or higher

  • Microsoft® Windows® XP Home, Professional, or Tablet PC Edition with Service Pack 2 or 3 (32-bit and 64-bit); Windows Server® 2003 (with Service Pack 2 for 64-bit); or Windows Vista® Home Basic, Home Premium, Business, Ultimate, or Enterprise with or without Service Pack 1 (32-bit and 64-bit)

·         MS Hot Fix KB-930627: http://support.microsoft.com/kb/930627 is required for Microsoft® Windows® XP Home, Professional, or Tablet PC Edition with Service Pack 2 (64-bit); Windows Server® 2003 (with Service Pack 2 for 64-bit); or Windows Vista® Home Basic, Home Premium, Business, Ultimate, or Enterprise without Service Pack 1 (64-bit)

·         256 MB of installed RAM (512 MB recommended)

·         90 MB of available hard disk space (Note: The installer creates temporary files during installation so the actual hard disk space required at instllation may be greater than the value listed.)

·         1024 x 768 screen resolution

·         Internet Explorer 6.0

 

Windows Installation Issues

 

This section contains issues you may encounter during installation on a Windows system.

64-bit Operating System Support

 

For supported 64-bit operating systems, MathCAD PDistiller requires the Microsoft KB930627 hot fix located at http://support.microsoft.com/kb/930627  in order to be installed successfully.

 

Installation on Vista Operating System

 

PROBLEM: Adobe PDF printer does not appear in the Control Panel under the scenario detailed below.

 

METHOD:

-           Install MathCAD pDistiller from command line - Adobe PDF gets installed and works ok;

-           Uninstall from Control Panel

-           Reboot computer

-           Install again to the same directory => Adobe PDF gets installed and works fine;

-           Uninstall;

-           DO NOT reboot, install again to the same directory.

 

RESULT: Adobe PDF printer does get installed and the user can print to the Adobe PDF printer without any problem but the Adobe PDF Printer doesn’t appear in the Control Panel.

 

WORKAROUND: Reboot after uninstall or clean up the printer registry. [1914893]

 

 

PROBLEM: The following warning is displayed during installation on Vista:

 

“User account control.

A program needs your permission to continue

Adobe PDF Creation Add-On9 Adobe Systems, Inc.,

Install 9.0.0

Adobe Systems

<Continue> <Cancel>”

 

WORKAROUND: Press Continue. Installation then proceeds. [1915365]
